# Apache JMeter Tutorials

Welcome to the **Apache JMeter Tutorials** repository! This project is a comprehensive collection of guides designed to help performance testing professionals and beginners master JMeter and the core principles of performance engineering.

## 📋 Table of Contents

### 1. Fundamentals of Performance Testing
* [Introduction to Performance Testing](PerformanceTestingIntroduction.html) - Core concepts and the importance of performance testing.
* [Types of Performance Testing](TypesOfPerformanceTesting.html) - Breakdown of Load, Stress, Endurance, and Spike testing.
* [Performance Testing Terminologies](PerformanceTestingTerminologies.html) - Key metrics like Latency, Throughput, and Response Time.
* [Performance Testing SLAs](PerformanceTestingSLAs.html) - Defining non-functional requirements and Service Level Agreements.

### 2. Strategy & Lifecycle
* [Performance Testing Life Cycle (PTLC)](PerformanceTestingLifeCycle.html) - The end-to-end process from planning to reporting.

* [Common Performance Problems](CommonPerformanceProblems.html) - Identifying bottlenecks, memory leaks, and high CPU usage.

### 3. Mastering Apache JMeter

* [JMeter Scripting Steps](ApacheJMeterScriptingSteps.html) - The logical flow for building any test plan.
* [Recording Scripts](RecordScriptsApacheJMeter.html) - How to use the HTTP(S) Test Script Recorder.
* [Test Elements Execution Order](TestElementsExecutionOrder.html) - Understanding the scoping and precedence of JMeter elements.

### 4. Execution & Results Analysis
* [Non-GUI Mode Commands](JMeterNon-GUIModeCommand.html) - Best practices for running tests via Command Line.
* [Performance Testing Monitoring](PerformanceTestingMonitoring.html) - Tracking server health during test execution.
